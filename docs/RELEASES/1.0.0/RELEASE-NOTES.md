# Release 1.0.0 — La Roulotte Solidaire Toulouse

Nous sommes fiers d’annoncer la première version stable de l’écosystème numérique de la Roulotte Solidaire Toulouse.

Cette version pose les fondations d’un environnement moderne, robuste et accessible pour soutenir les actions de solidarité sur le terrain.

## Points clés
- Scripts de maintenance avancés (Bash + Python)
- Pack utilitaire “La Route”
- API Ruby v1
- Documentation initiale
- Structure de projet stabilisée

Les versions 1.1.0 et 1.2.0 sont déjà en préparation.
