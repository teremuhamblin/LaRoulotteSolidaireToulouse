🚐✅ Checklists Opérationnelles — Maraude

1️⃣ Avant la maraude — Préparation

👥 Équipe
- [ ] Responsable de Tournée désigné  
- [ ] Conducteur identifié  
- [ ] Référent Distribution  
- [ ] Référent Lien Social  
- [ ] Gestionnaire des Stocks  
- [ ] Bénévoles présents et briefés  

📦 Matériel
- [ ] Trousse de premiers secours  
- [ ] Téléphones chargés  
- [ ] Lampes frontales  
- [ ] Gants, gel hydroalcoolique  
- [ ] Couvertures de survie  
- [ ] Carnet de bord / fiches  
- [ ] Flyers d’information (hébergement, droits, numéros utiles)

🍲 Stocks
- [ ] Boissons chaudes / froides  
- [ ] Nourriture  
- [ ] Kits hygiène  
- [ ] Vêtements / couvertures  
- [ ] Accessoires saisonniers (bonnets, eau, crème solaire…)

🚐 Véhicule
- [ ] Carburant OK  
- [ ] Pneus OK  
- [ ] Éclairage OK  
- [ ] Propreté OK  
- [ ] Matériel chargé et accessible  

🗺️ Parcours
- [ ] Points de maraude validés  
- [ ] Horaires définis  
- [ ] Zones sensibles identifiées  
- [ ] Adaptations éventuelles faites  

---

2️⃣ Pendant la maraude — Exécution

🧭 Déplacements
- [ ] L’équipe reste groupée  
- [ ] Aucun bénévole isolé  
- [ ] Zones à risque évitées  
- [ ] Véhicule toujours accessible  

🤝 Interactions
- [ ] Approche calme et respectueuse  
- [ ] Distance de sécurité respectée  
- [ ] Écoute active  
- [ ] Pas de promesses impossibles  
- [ ] Respect du refus d’aide  

📊 Suivi
- [ ] Quantités distribuées notées  
- [ ] Points effectués cochés  
- [ ] Situations sensibles signalées au RT  
- [ ] Personnes vulnérables repérées  

🛡️ Sécurité
- [ ] Tensions gérées calmement  
- [ ] Retrait si danger  
- [ ] Appel aux secours si nécessaire  
- [ ] Aucun contact physique dans un conflit  

---

3️⃣ Après la maraude — Clôture

🧹 Rangement
- [ ] Matériel rangé  
- [ ] Véhicule nettoyé  
- [ ] Stocks restants notés  
- [ ] Déchets triés  

📝 Débriefing
- [ ] Points positifs  
- [ ] Difficultés rencontrées  
- [ ] Situations sensibles à suivre  
- [ ] Ressenti des bénévoles  

📘 Carnet de bord
- [ ] Nombre de personnes rencontrées  
- [ ] Besoins identifiés  
- [ ] Incidents éventuels  
- [ ] Informations transmises à l’équipe suivante
