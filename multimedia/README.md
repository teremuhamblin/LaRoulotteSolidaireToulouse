🎨 Dossier MULTIMÉDIA
Photos, vidéos, logos, visuels et supports graphiques de La Roulotte Solidaire Toulouse

Ce dossier regroupe l’ensemble des contenus visuels utilisés par La Roulotte Solidaire Toulouse :  
photos, vidéos, logos, affiches, illustrations, contenus réseaux sociaux et autres éléments graphiques.

Il constitue la base multimédia officielle du projet.

---

🗂️ Organisation du dossier

Le dossier est structuré pour rester clair, propre et facile à maintenir :

`
MULTIMÉDIA/
├── photos/              # Photos des actions, matériel, événements (sans visages identifiables)
├── videos/              # Vidéos courtes, témoignages anonymisés, clips de présentation
├── logos/               # Logos officiels, variantes, formats PNG/SVG
├── affiches/            # Affiches, flyers, supports imprimables
└── reseaux_sociaux/     # Visuels optimisés pour Instagram, Facebook, etc.
`

Chaque sous-dossier contient un README dédié si nécessaire.

---

🧩 Règles d’utilisation des médias

✔️ Ce que nous acceptons
- photos de matériel, actions, distributions  
- visuels de communication  
- affiches, flyers, bannières  
- logos officiels  
- vidéos anonymisées  
- illustrations et créations graphiques  

❌ Ce que nous n’acceptons pas
> Aucun visage identifiable ne doit apparaître, sauf consentement écrit explicite.  
> Aucune donnée personnelle ne doit être visible (nom, lieu précis, situation sensible).  
> Aucune image dégradante, stigmatisante ou intrusive.

Ces règles protègent la dignité des personnes rencontrées.

---

📝 Nommage des fichiers

Pour garder une bibliothèque propre et durable, merci de respecter ce format :

`
annéemoisdescription_version.extension
`

Exemples :

- 202403collecteaffichev1.png
- 202402maraudephotosmateriel_v2.jpg
- 202401logoofficielcouleur.svg

---

🎨 Logos & identité visuelle

Les logos officiels se trouvent dans :

`
MULTIMÉDIA/logos/
`

Ils doivent être utilisés :

- sans modification non validée  
- dans le respect des couleurs et proportions  
- uniquement pour des supports liés au projet  

Si tu veux, une charte graphique peut être ajoutée ici.

---

📤 Ajout de nouveaux médias

Pour ajouter un média :

1. Vérifier qu’il respecte les règles éthiques  
2. Le placer dans le bon sous-dossier  
3. Utiliser un nom de fichier clair  
4. Ajouter une courte description dans un fichier README.md si nécessaire  
5. Ouvrir une pull request si le dépôt est collaboratif  

---

🔐 Respect de la vie privée

La Roulotte Solidaire Toulouse s’engage à :

- protéger la dignité des personnes  
- ne jamais exposer de situations sensibles  
- ne jamais publier de visages sans accord  
- anonymiser les contenus si nécessaire  

Toute image non conforme sera retirée immédiatement.

---

🧰 Formats recommandés

- Images : PNG, JPG, SVG  
- Vidéos : MP4, MOV  
- Documents graphiques : PDF, PNG haute résolution  
- Logos : SVG (vectoriel), PNG transparent
