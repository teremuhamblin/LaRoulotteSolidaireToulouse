# Roadmap — La Roulotte Solidaire Toulouse

## Version 1.1.0 — Stabilité & Automatisation
Objectif : fiabiliser, automatiser, fluidifier.

### Contenu prévu
- Automatisation CRON des scripts de maintenance
- API v1.1 (endpoints supplémentaires)
- Intégration des rapports JSON dans l’API
- Mode hors‑ligne pour le pack “La Route”
- Documentation utilisateur + guide bénévole numérique
- Script de synchronisation Git automatique

## Version 1.2.0 — Expérience & Accessibilité
Objectif : rendre tout plus simple, plus beau, plus accessible.

### Contenu prévu
- Interface TUI Textual (Python)
- Dashboard HTML local pour visualiser les rapports
- API v2 (auth renforcée, pagination, filtres)
- Notifications Telegram/Discord
- Documentation illustrée (MULTIMÉDIA)
- Mode “démo” pour les nouveaux bénévoles
